# FitHelper

FitHelper is een Python console applicatie waarmee een gebruiker fitnessgegevens kan bijhouden.

## Functionaliteiten

- Profiel aanmaken
- Profiel bekijken
- Workout toevoegen
- Workouts bekijken
- Workout verwijderen
- Progressie bekijken
- Gewicht toevoegen
- Gewicht geschiedenis bekijken

## Gebruikte technieken

- Python
- Functies
- Error handling
- JSON bestanden
- Unit tests met pytest

## Project starten

```bash
python FitHelper/main.py
